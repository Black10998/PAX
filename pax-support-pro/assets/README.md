# WordPress.org Assets

This directory contains assets for WordPress.org plugin directory listing.

## Required Files

- `banner-772x250.png` - Plugin banner (low resolution)
- `banner-1544x500.png` - Plugin banner (high resolution)  
- `icon-128x128.png` - Plugin icon (low resolution)
- `icon-256x256.png` - Plugin icon (high resolution)
- `screenshot-1.png` - Chat interface screenshot
- `screenshot-2.png` - Admin settings screenshot
- `screenshot-3.png` - Callback scheduling screenshot

## Specifications

### Banners
- Low res: 772x250px
- High res: 1544x500px
- Format: PNG or JPG
- Max size: 1MB

### Icons
- Low res: 128x128px
- High res: 256x256px
- Format: PNG
- Max size: 256KB

### Screenshots
- Recommended: 1280x720px or higher
- Format: PNG or JPG
- Max size: 1MB each

## Notes

These assets are used by WordPress.org and should showcase the plugin's key features and professional design.
